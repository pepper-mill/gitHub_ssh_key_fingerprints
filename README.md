# gitHub-s_ssh_key_fingerprints
Public key fingerprints that can be used to validate a connection to a remote server.

## Source
- https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/githubs-ssh-key-fingerprints

# Directions
- Clone the inlcuded know_hosts file to your ~/.ssh directory to avoid manually verifying GitHub hosts. It contains the ssh public key entries.
