# gitHub-s_ssh_key_fingerprints
Public key fingerprints that can be used to validate a connection to a GitHub remote server.

## Source
- https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/githubs-ssh-key-fingerprints

# Directions
- Add the known_hosts file's contents to your local ~/.ssh/known_hosts file to avoid manually verifying GitHub hosts.
